20 Most Filmed Locations In The US

On this map you'll find the 20 most filmed locations within the United States over the past 20 years. This list was gathered from GoCompare.com's On Location project which you can find here. As you can see all locations were either in California or New York. Zoom in to see the names of these locations and how many movies or shows were shot there.

This page was created by Emma Ross in 2020. The page is for a project using Leaflet and MapBox. You can click here to view the github repo.
